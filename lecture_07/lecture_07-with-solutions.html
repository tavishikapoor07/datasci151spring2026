Out of the many built-in objects in Python, two of them are of "Boolean" type. These are the objects True and False

These should not be confused with the strings "True" and "False"

What is a Boolean object? It is just an object that can take one of two possible values. In this case, True and False.

We can assign these values to variables directly:

In[3]:

But more commonly, we obtain Boolean values in other, indirect ways.

II. Comparison Operators

  • One of the most common ways Boolean objects show up is in the use of comparison operators.

  • These are operators used to compare two values, and evaluate to either True or False.

  • There are six basic comparison operators. The first four are:

    • > - Greater than

    • < - Less than

    • >= - Greater than or equal to

VII. Control Statements - if, elif, else

  • Used to adapt the code to different logical conditions

  • Very helpful when loading, analyzing, and cleaning data!

Running a part of the code
only if a condition is met: 
    if test_expression:
        <body of if>

Test expression

  • We type if followed by a logical condition and the : (colon) symbol.
  • The : indicates that an indented code block is starting

Body of expression

  • The "body" of the if statement needs to be indented
  • You can indent text by pressing the "tab" button on your keyboard.

If the condition is true, the code in the indented block will be run.

If the condition is false, then the indented block is skipped.

13) Can you think of a better way of checking if two floating-point numbers are "equal"? Code up your approach below

Answer: It is generally not advisable to check if two floating point numbers are equal. Instead, check that the absolute value of their difference is smaller than some "tolerance" value.

$$ |x - y| < \textrm{tolerance}$$

In[]:

y = 0.3

# Choose a tolerance of 10^(-12) = 0.000000000001
tolerance = 10**(-12)
tolerance_alternative = 1e-12 # another way of writing 10^(-12)

print(abs(x - y) < tolerance)
print(abs(x - y) < tolerance_alternative)
